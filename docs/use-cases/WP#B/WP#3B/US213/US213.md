# US213 - Find Scheduled Flights by Aircraft

## 1. User Story
> As a Backoffice Operator, I want to list all scheduled flights assigned to a specific aircraft to audit its utilization schedule.

## 2. Acceptance Criteria
- The request must provide a valid `aircraftRegistration` path parameter.
- The system must query and retrieve all matching scheduled flights sorted chronologically.
- On success, returns HTTP `200 OK` with the collection array.

## 3. Pre-conditions
- The actor must be authenticated and authorized with operational privileges.

## 4. Post-conditions
- Read-only data model query lookup operation; no structural state alteration.

## 5. Main Success Scenario
1. The actor sends a `GET /api/scheduled-flights/aircraft/{registration}` request.
2. The `ScheduledFlightController` extracts the resource parameter target.
3. The `GetScheduledFlightsByAircraftUseCase` issues an evaluation fetch on the repository layer.
4. The database extracts rows matching the registration constraint.
5. The system serves an HTTP `200 OK` collection mapping.

## 6. Alternative / Exception Flows
| Step | Condition | System Response |
|------|-----------|-----------------|
| 1 | Context token validation fails | HTTP `401 Unauthorized` |
| 3 | The target aircraft registration does not exist | HTTP `404 Not Found` |

## 7. Design Justification
- **Query Isolation:** Dedicated read endpoints protect performance by avoiding fetching the absolute flight schedule array when looking up an isolated vehicle stream context.
