# Glossário do Sistema AlSafe

| Conceito | Estereótipo DDD | Descrição                                                                                                                                                                                                                         |
| :--- | :--- |:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **User** | Aggregate Root / Entity | Utilizador do sistema, que contém as credenciais de acesso seguras.                                                                                                                                                               |
| **Role** | Value Object | Papel de autorização (ex: Backoffice Operator, ATCC) que define os acessos permitidos ao utilizador.                                                                                                                              |
| **Manufacturer** | Aggregate Root / Entity | Empresa fabricante de aeronaves, pré-carregada no arranque do sistema.                                                                                                                                                            |
| **AirportType** | Aggregate Root / Entity | Categoria de um aeroporto (ex: Internacional, Regional), pré-carregada no arranque do sistema.                                                                                                                                    |
| **AircraftModel** | Aggregate Root / Entity | Especificações técnicas de fábrica de uma aeronave. Inclui alcance, documentação/diagramas técnicos e fabricante, mas **não** a configuração final de lugares.                                                                    |
| **Aircraft** | Aggregate Root / Entity | Aeronave física pertencente à frota. Regista a matrícula, a configuração real de lugares (*seatingCapacity*), o total de horas de voo (*totalFlightHours*) e o cálculo derivado de horas operacionais (*/totalOperationalHours*). |
| **AircraftStatus** | Enum / Value Object | Estado operacional de uma aeronave num dado momento (Available, In-Flight, Under Maintenance, Inactive).                                                                                                                          |
| **Airport** | Aggregate Root / Entity | Infraestrutura aeroportuária (IATA). Regista contactos, horário de funcionamento e certifica modelos de aviões compatíveis com as suas pistas.                                                                                    |
| **AirportStatus** | Enum / Value Object | Estado de funcionamento de um aeroporto (Operational, Closed, Under Maintenance).                                                                                                                                                 |
| **Coordinates** | Value Object | Localização geográfica exata (latitude/longitude) pertencente a um aeroporto.                                                                                                                                                     |
| **RunwayInfo** | Value Object | Dados estruturais de uma pista (comprimento, orientação) pertencente a um aeroporto.                                                                                                                                              |
| **FacilityInfo** | Value Object | Agrupa a informação detalhada sobre as infraestruturas de passageiros num aeroporto (número de terminais, portas de embarque e serviços).                                                                                         |
| **Route** | Aggregate Root / Entity | Rota de voo planeada no sistema (identificada por ID), ligando um aeroporto de origem a um de destino.                                                                                                                            |
| **RouteRequirements** | Value Object | Exigências mínimas técnicas (alcance e capacidade) exigidas a um avião para poder operar uma Rota.                                                                                                                                |
| **ScheduledFlight** | Aggregate Root / Entity | Representa a alocação temporal e real de uma aeronave física a uma Rota para realizar um voo num dia e hora específicos.                                                                                                          |
| **FlightStatus** | Enum / Value Object | Lista finita de estados possíveis para o ciclo de vida de um voo agendado (Scheduled, Delayed, Canceled, In-Flight, Completed).                                                                                                   |
| **MaintenanceTemplate** | Aggregate Root / Entity | Molde base de manutenção, contendo um tipo e uma checklist imutável de tarefas a verificar.                                                                                                                                       |
| **TemplateType** | Enum / Value Object | Categoria do template de manutenção (Inspection, Scheduled Maintenance, Overhaul, Modification).                                                                                                                                  |
| **Checklist** | Value Object | Lista de tarefas de verificação embutida num Template de Manutenção.                                                                                                                                                              |
| **MaintenanceRecord** | Aggregate Root / Entity | Registo histórico e financeiro de uma intervenção efetuada numa aeronave. Inclui a duração, os custos financeiros e o tipo de componente intervencionado.                                                                         |
| **ComponentType** | Enum / Value Object | Lista finita que categoriza a parte física da aeronave afetada pela manutenção (Engine, Airframe, Avionics, Interior, Exterior).                                                                                                  |
